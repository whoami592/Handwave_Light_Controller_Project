Simple USE
To use this HandWave Light Controller:
Install required libraries: pip install opencv-python numpy pyserial
Connect an Arduino or similar device to control lights via serial port (modify 'COM3' in code to match your setup)
Wave your hand in front of the webcam to detect gestures
Two consecutive waves within 2 seconds toggle the light state
Press 'q' to quit the program
